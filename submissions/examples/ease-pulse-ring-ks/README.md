# Pulse Ring Hover

## What does this do?
Triggers a repeating, expanding, and fading ripple outline effect (pulse ring) when hovering over a button or interactive element.

## How is it used?
Add the class to your button or link:

```html
<button class="pulse-ring-btn">Hover over me</button>
```

## Why is it useful?
It provides dynamic, modern visual feedback for calls-to-action (CTAs) and high-priority buttons without requiring any JavaScript or heavy dependencies.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
