# Float & Blur Shadow

## What does this do?
Simulates a floating/lifting effect on hover by translating the element upwards while increasing the shadow's blur radius and spread.

## How is it used?
Add the class to your card or container:

```html
<div class="float-shadow-card">
  <h3>Card Title</h3>
  <p>Content goes here.</p>
</div>
```

## Why is it useful?
It creates physical depth in the user interface, improving interactivity feedback for cards, grid items, and modular containers.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
